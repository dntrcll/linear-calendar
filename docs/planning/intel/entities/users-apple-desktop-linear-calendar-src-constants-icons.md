# icons.js

## Purpose

Provides a centralized collection of SVG icon components used throughout the application. Each icon is a React functional component that accepts props for customization.

## Exports

- **ICONS**: Object containing icon components (Settings, Trash, Plus, ChevronLeft, ChevronRight, Close, Calendar, Clock, MapPin)
- **default**: The ICONS object (default export)

## Notes

All icons are implemented as inline SVG components with consistent styling patterns (stroke-based, configurable via props). Default dimensions and stroke widths vary by icon to maintain visual consistency.
